# KB904 产品配置

## 产品概述

| 属性 | 值 |
|------|-----|
| **型号** | KB904 |
| **产品名** | inateck NeoMagic Pro |
| **主控** | WCH CH584M (RISC-V) |
| **功能** | 蓝牙键盘 + 触控板 |

## 配置特点

### 与默认配置的差异

| 配置项 | 默认值 | KB904 值 |
|--------|--------|----------|
| 蓝牙通道数 | 3 | 1 |
| 背光颜色数 | 7 | 13 |
| 背光类型 | 单色 | RGB |
| 指示灯数量 | 2 | 4 |
| 深度休眠 | 10分钟 | 10分钟（同默认） |

### 功能开关

| 功能 | 状态 |
|------|------|
| 蓝牙 | ✅ 启用 |
| USB | ❌ 禁用 |
| 触控板 | ✅ 启用 |
| 组合键 | ✅ 启用 |

## 硬件配置

### 矩阵

- **行数**: 8
- **列数**: 16
- **二极管方向**: COL2ROW
- **总键数**: 128（实际使用约 80 键）

### 背光

- **类型**: RGB 三色
- **颜色数**: 13
- **亮度档位**: 4（无/1/2/3）
- **休眠超时**: 5秒无操作熄灭

### 指示灯

| LED | 引脚 | 颜色 | 功能 |
|-----|------|------|------|
| CAPS | PA15 | 白 | 大写锁定 |
| BT | PB22 | 蓝 | 蓝牙状态 |
| POWER | PB23 | 红 | 电源/低电量 |
| CHARGE | PA14 | 绿 | 充满电（只读）|

### 触控板

- **芯片**: PCT1336QN
- **接口**: I2C (PB20/PB21)
- **中断**: PA3（下降沿触发）
- **分辨率**: 2048 × 1024

## 文件结构

```
keyboards/kb904/
├── config.h        # 产品主配置（覆盖默认值）
├── config_hw.h     # GPIO 硬件映射（对应原理图）
└── README.md       # 本文件
```

## 配置加载顺序

1. `config.h` - 产品配置（定义覆盖项）
2. `config_hw.h` - GPIO 映射
3. `defaults/*.h` - 默认值（通过 #ifndef 保护）
4. `config_check.h` - 编译时验证
