  第三部分：电源管理模块设计

  电源管理是 N0046 的核心模块，需要新增 drivers/power/power_switch.c：

  状态机设计：
  [断电] --电源键按下--> [上电检测]
  [上电检测] --长按3秒--> [开机运行] (拉高B19)

  [开机运行] --10分钟无操作--> [深度休眠]
  [深度休眠] --任意键按下--> [开机运行]

  关键函数：
  void power_switch_init(void);           // 初始化电源检测
  void power_switch_task(void);           // 主循环中调用，检测电源键
  void power_shutdown(void);              // 执行关机（拉低B19）
  void power_enter_sleep(void);           // 进入深度休眠
  void power_exit_sleep(void);            // 退出休眠
  bool power_button_pressed(void);        // 电源键状态

  1. 硬件上电时 B19 默认低电平
  2. 固件启动后立即检测 A2（电源键）
  3. 若持续按住 3 秒，拉高 B19 保持供电，进入正常运行
  4. 若中途松开，不拉高 B19，芯片自动断电

  │ 回连中   │ 慢闪 (1次/秒) │ 开机有配对记录/断连后轮询            │

      bool esc_pressed = matrix_is_on(ESC_ROW, ESC_COL);
      bool equal_pressed = matrix_is_on(EQUAL_ROW, EQUAL_COL);

      if (esc_pressed && equal_pressed) {
          if (reset_combo_start == 0) {
              reset_combo_start = timer_read32();
          } else if (timer_elapsed32(reset_combo_start) >= 3000) {

      }

  第三部分：电源管理模块设计

  电源管理是 N0046 的核心模块，需要新增 drivers/power/power_switch.c：

  状态机设计：
  [断电] --电源键按下--> [上电检测]
  [上电检测] --长按3秒--> [开机运行] (拉高B19)
  [上电检测] --短按松开--> [断电] (不拉高B19，自动断电)
  [开机运行] --长按电源键3秒--> [关机] (拉低B19)
  [开机运行] --单按电源键--> [发送锁屏键码]
  [开机运行] --10分钟无操作--> [深度休眠]
  [深度休眠] --任意键按下--> [开机运行]

  关键函数：
  void power_switch_init(void);           // 初始化电源检测
  void power_switch_task(void);           // 主循环中调用，检测电源键
  void power_shutdown(void);              // 执行关机（拉低B19）
  void power_enter_sleep(void);           // 进入深度休眠
  void power_exit_sleep(void);            // 退出休眠
  bool power_button_pressed(void);        // 电源键状态

  开机流程：

      bool esc_pressed = matrix_is_on(ESC_ROW, ESC_COL);
      bool equal_pressed = matrix_is_on(EQUAL_ROW, EQUAL_COL);

      if (esc_pressed && equal_pressed) {
          if (reset_combo_start == 0) {
              reset_combo_start = timer_read32();
          } else if (timer_elapsed32(reset_combo_start) >= 3000) {
              factory_reset_execute();
              reset_combo_start = 0;
          }
      } else {

      }
