# KB04122 键盘产品功能清单

## 1. 项目概述

| 项目 | 规格 |
|------|------|
| **产品型号** | KB04122 |
| **连接方式** | 蓝牙 5.3+（单通道） |
| **支持系统** | iPad OS |
| **键盘布局** | US 布局 |
| **丝印语种** | US、DE、UK、JP、FR、IT、ES |
| **触控板** | 双按钮，支持全手势 |
| **背光** | 无 |
| **主控芯片** | 待定 |
| **电池** | 3.7V 500mAh 锂电池 |
| **蓝牙配对名称** | Inateck KB04122 |

## 4. 蓝牙模块

### 4.1 蓝牙基础参数

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 蓝牙版本 | 蓝牙 5.3 及以上 | P0 | 硬件 |
| 配对名称 | "Inateck KB04122" | P0 | 固件 |
| 有效距离 | 10 米范围内有效操作 | P0 | 硬件 |
| 单通道连接 | 仅支持连接 1 台设备 | P0 | 固件 |

### 4.2 蓝牙配对功能

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 首次配对 | 首次开机或恢复出厂后自动进入配对模式 | P0 | 固件 |
| 手动配对 | 长按 Fn+Del 3 秒启动配对模式 | P0 | 固件 |
| 配对超时 | 广播 60 秒，超时进入休眠 | P0 | 固件 |
| 配对完成时间 | 设备端点击配对后 3 秒内完成连接 | P0 | 固件 |
| 配对取消处理 | 用户取消配对弹窗后重新进入配对状态 | P1 | 固件 |

### 4.3 蓝牙回连/轮询功能

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 开机自动回连 | 开机后直接回连上次连接的设备，无需按键 | P0 | 固件 |
| 回连完成时间 | 3 秒内完成回连 | P0 | 固件 |
| 回连超时处理 | 回连失败后进入轮询模式（可接受新连接） | P0 | 固件 |
| 轮询模式时长 | 轮询 60 秒，超时进入休眠 | P0 | 固件 |
| 休眠唤醒回连 | 按任意键唤醒后自动回连 | P0 | 固件 |
| 断连后轮询 | 设备关机/关蓝牙/超距后进入轮询模式 | P0 | 固件 |
| 配对模式后重启 | 配对模式中关机重启后回连上次设备 | P1 | 固件 |

### 4.4 蓝牙休眠与功耗

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 深度休眠进入 | 蓝牙断开后 10 分钟进入深度休眠 | P0 | 固件 |

## 5. 指示灯模块

### 5.1 指示灯硬件定义

| 位置 | 编号 | 颜色 | 用途 |
|------|------|------|------|
| 左侧 | 指示灯1 | 白色 | 大写锁定指示灯 |
| 中间 | 指示灯2 | 蓝色 | 蓝牙状态指示灯 |
| 右侧 | 指示灯3 | 绿色/红色 | 电量指示灯、充电指示灯 |

### 5.2 灯效定义

| 灯效名称 | 描述 | 用途 |
|----------|------|------|
| 慢闪 | 每秒闪 1 次 | 回连状态 |
| 快闪 | 每秒闪 2 次 | 配对状态 |
| 常亮 | 持续点亮 | 状态确认 |

### 5.3 大写锁定指示（指示灯1 - 白灯）

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 大写锁定亮起 | 大写锁定时白灯常亮 | P0 | 固件 |
| 大写未锁定熄灭 | 大写未锁定时白灯熄灭 | P0 | 固件 |

### 5.4 蓝牙状态指示（指示灯2 - 蓝灯）

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 已连接状态 | 蓝牙已连接时蓝灯熄灭 | P0 | 固件 |
| 未连接状态 | 蓝牙未连接时蓝灯熄灭 | P0 | 固件 |
| 配对状态 | 配对模式时蓝灯快闪（2次/秒） | P0 | 固件 |
| 回连状态 | 回连模式时蓝灯慢闪（1次/秒） | P0 | 固件 |

### 5.5 电量/充电指示（指示灯3 - 红灯/绿灯）

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 正常电量（开机） | 电量 20%~100% 时熄灭 | P0 | 固件 |
| 低电量警告 | 电量 0%~20% 时红灯闪烁 | P0 | 固件 |
| 充电中 | 充电未满时红灯常亮 | P0 | 固件 |
| 充满 | 充电完成时绿灯常亮 | P0 | 固件 |
| 关机充电 | 关机状态下显示充电状态 | P0 | 固件 |
| 开机指示 | 开机时电量指示灯闪烁 2.5 秒 | P1 | 固件 |
| 电量查询 | Fn+右Cmd 后红灯闪烁显示电量档位（1~4次对应25%档位） | P1 | 固件 |

### 5.6 充电时休眠行为

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 已配对充电不休眠 | 蓝牙已配对时，充电状态下不进入休眠 | P1 | 固件 |
| 未配对充电可休眠 | 蓝牙未配对时，充电状态下可进入休眠，按键唤醒进入广播 | P1 | 固件 |

### 5.7 恢复出厂指示

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 出厂设置完成 | 恢复出厂后所有指示灯同时闪烁 3 次 | P1 | 固件 |
| 进入配对模式 | 恢复出厂后指示灯2蓝灯闪烁 | P0 | 固件 |

| 优先级 | 数量 | 说明 |
