## 1. 目标

实现 N0046 键盘的 US 布局键位映射，确保所有按键正确识别和发送。

| 列数 | 16 |

## 3. 键位映射要求

### 3.1 Layer 0（默认层）

| 按键 | HID 键码 | 说明 |
|------|----------|------|
| ESC | KC_ESC | 退出 |
| F1-F12 | KC_F1-KC_F12 | 功能键 |
| Del | KC_DEL | 删除 |
| 字母键 | KC_A-KC_Z | 字母输入 |
| 数字键 | KC_1-KC_0 | 数字输入 |
| Space | KC_SPC | 空格 |
| Enter | KC_ENT | 回车 |
| Shift | KC_LSFT/KC_RSFT | 左/右 Shift |
| Ctrl | KC_LCTL/KC_RCTL | 左/右 Ctrl |
| Alt | KC_LALT/KC_RALT | 左/右 Alt |
| GUI | KC_LGUI/KC_RGUI | 左/右 GUI (Cmd) |
| Caps | KC_CAPS | 大写锁定 |

### 3.2 Layer 1（Fn 层）

| 按键 | HID 键码 | 功能 |
|------|----------|------|
| F1 | KC_ESC | ESC（防止误触） |
| F2 | KC_BRID | 屏幕亮度减 |
| F3 | KC_BRIU | 屏幕亮度加 |
| F4 | KC_FIND | 搜索 |
| F5 | KC_PSCR | 截屏 |
| F6 | KC_SIRI | Siri（自定义键码） |
| F7 | KC_MPRV | 上一首 |
| F8 | KC_MPLY | 播放/暂停 |
| F9 | KC_MNXT | 下一首 |
| F10 | KC_MUTE | 静音 |
| F11 | KC_VOLD | 音量减 |
| F12 | KC_VOLU | 音量加 |
| Del | RESET | 进入配对模式（需长按） |

## 6. 实现要求

### 6.1 键位数组格式

使用项目现有的 `KEYMAP` 宏定义：

```c
const uint16_t keymaps[][MATRIX_ROWS][MATRIX_COLS] = {
    [0] = KEYMAP(
        // Row 0: ESC, F1-F12, Del
        KC_ESC,  KC_F1,   KC_F2,   ...
        // Row 1-5: 主键盘区
        ...
        // Row 6-7: 扩展键（如有）
        ...
    ),
    [1] = KEYMAP(
        // Fn 层
        ...
    )
};
```

### 6.2 矩阵位置映射

**需要根据实际原理图确定每个物理键的行列位置。**

参考 `docs/N0046_原理图.md` 中的 FPC1 连接器定义：
- Pin 1-8: R0-R7 (行线)
- Pin 9-26: C0-C15 (列线，非连续编号)
