## 5. 文件变更总览

### 5.1 新增文件 (3 个)

| 文件路径 | 职责 |
|----------|------|
| `keyboards/kb904/config.h` | 产品配置 |

### 5.2 修改文件 (9 个)

| 文件路径 | 修改内容 |
|----------|----------|
| `keyboards/indicator_config.h` | 条件编译支持 4 灯 |
| `drivers/output/backlight/backlight.h` | 13 色枚举 |
| `drivers/output/backlight/backlight.c` | 13 色预设表 |
| `middleware/keyboard/combo/kb_combo_map.c` | 新增组合键 |
| `middleware/keyboard/combo/kb_fn_action.c` | 新增 Action |

| `drivers/power/battery.c` | 复用 IND_REQ 机制 |
| `application/service/output_service.h` | 背光休眠接口 |
| `application/service/output_service.c` | 背光休眠 + 低电量联动 |

## 8. GPIO 映射

| 功能 | GPIO | 说明 |
|------|------|------|
| 背光 R | PB0 | PWM |
| 背光 G | PA12 | PWM |
| 背光 B | PA13 | PWM |
| 大写指示灯 (白) | A15 | GPIO |
| 蓝牙指示灯 (蓝) | B22 | GPIO |
| 电源指示灯 (红) | B23 | GPIO |
| 充满指示灯 (绿) | A14 | GPIO (只读) |

## 9. 验收标准

### 9.1 功能验收

- [ ] 13 色背光切换正常
- [ ] 4 档亮度切换正常
- [ ] 5 秒背光休眠正常
- [ ] 按键唤醒背光正常
- [ ] 低电量时背光关闭
- [ ] 所有 Fn 组合键正常
- [ ] 电量检查闪烁 1-4 次
- [ ] 恢复出厂设置正常

### 9.2 性能验收

- [ ] 背光唤醒延迟 <50ms
- [ ] 电量检测精度 ±5%
