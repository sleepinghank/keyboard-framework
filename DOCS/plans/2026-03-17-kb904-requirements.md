# KB904 蓝牙妙控键盘固件需求摘要

> 文档版本: V1.0
> 生成日期: 2026-03-17
> 基于: KB904_功能清单.md + Agent 评审

---

## 1. 产品概述

| 项目 | 规格 |
|------|------|
| **产品型号** | KB904 蓝牙妙控键盘 |
| **配对名称** | inateck NeoMagic Pro |
| **连接方式** | 蓝牙 BLE 5.3（单通道） |
| **目标系统** | iPad（默认） |
| **主控芯片** | WCH 584M |
| **触控芯片** | Pixart PCT1336QN |
| **电池** | 500mAh / 3.7V 锂电池 |

---

## 2. 交付范围

| 优先级 | 数量 | 状态 |
|--------|------|------|
| P0（核心功能） | 42 项 | ✅ 必须实现 |
| P1（重要功能） | 17 项 | ✅ 必须实现 |
| P2（增值功能） | 2 项 | ❌ 本次不实现 |

**明确排除的功能**：
- OTA 固件升级
- ZWSN 手势自定义系统（使用 iPad 原生 PTP）
- APP 配置接口（背光时间/休眠时间调整）
- 霍尔开关固件处理（纯硬件控制）

---

## 3. 已确认的功能需求

### 3.1 系统兼容性

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 蓝牙连接与打字 | 支持蓝牙连接并正常打字输入 | P0 |
| 快捷键支持 | 支持多媒体及系统快捷键 | P0 |
| 触摸板全手势 | 支持 iPad 全部原生触控板手势（PTP 模式） | P0 |

### 3.2 键芯矩阵

| 功能 | 触发方式 | 描述 | 优先级 |
|------|----------|------|--------|
| 蓝牙配对 | Fn+Del 长按 3s | 启动蓝牙配对模式 | P0 |
| 恢复出厂设置 | ESC+= 长按 3s | 清除配对记录，恢复默认设置 | P0 |
| 电量检查 | Fn+右Cmd | 指示灯闪烁显示电量档位 | P1 |
| 切换背光亮度 | Fn+右Shift | 循环切换 4 档亮度 | P1 |
| 切换背光颜色 | Fn+右Enter | 循环切换 13 种颜色 | P1 |
| 自定义键 | 单按 | 固定功能：唤起 Siri（Consumer 0xCF） | P1 |

### 3.3 多媒体/系统快捷键

| 功能 | 按键 | 优先级 |
|------|------|--------|
| 上一曲/下一曲/播放暂停 | 多媒体键 | P0 |
| 静音/音量+/音量- | 多媒体键 | P0 |
| 亮度+/亮度- | 系统键 | P1 |
| 搜索/截屏/锁屏 | 系统键 | P1 |

### 3.4 背光系统

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 13 色切换 | 红-暗红-粉红-紫-靛青-深蓝-蓝-青蓝-青绿-绿-淡黄-橙-白 | P1 |
| 4 档亮度 | 无-一档-二档-三档 | P1 |
| 默认值 | 开机默认白光 2 档 | P1 |
| 5s 休眠 | 5 秒无操作背光熄灭，按键亮起 | P1 |
| 颜色记忆 | 关机/休眠后保持上次颜色设置 | P1 |
| 低电量关闭 | 电量 ≤20% 时关闭背光 | P1 |
| 调色限制 | 只有背光亮起时才能调节颜色 | P1 |

### 3.5 蓝牙连接

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 首次配对 | 首次开机或恢复出厂后自动进入配对模式 | P0 |
| 手动配对 | 长按 Fn+Del 3s 启动配对模式 | P0 |
| 配对超时 | 广播 60s，超时进入休眠 | P0 |
| 开机回连 | 开机后直接回连上次连接的设备 | P0 |
| 唤醒回连 | 休眠唤醒后自动回连 | P0 |
| 回连超时 | 回连 60s，超时进入休眠 | P0 |
| 深度休眠 | 10 分钟无操作进入深度休眠 | P0 |
| 触控唤醒 | 休眠状态下触摸板轻触立即响应 | P0 |

### 3.6 指示灯系统

#### 3.6.1 大写锁定指示灯（白灯）

| 状态 | 灯效 |
|------|------|
| 大写锁定 | 白灯常亮 |
| 大写未锁定 | 熄灭 |

#### 3.6.2 蓝牙状态指示灯（蓝灯）

| 状态 | 灯效 |
|------|------|
| 已连接 | 熄灭 |
| 未连接 | 熄灭 |
| 配对中 | 快闪（3Hz） |
| 回连中 | 慢闪（1Hz） |

#### 3.6.3 电源指示灯（红/绿灯）

| 场景 | 状态 | 灯效 |
|------|------|------|
| 电量 100%~20% | 正常 | 熄灭 |
| 电量 <20% | 低电量警告 | 红灯闪烁 |
| 充电中 | 未充满 | 红灯常亮 |
| 充电中 | 已充满 | 绿灯常亮 |
| 充满后 | 仍低电 | 继续闪烁 |
| 电量检查 | Fn+右Cmd | 1-4 次闪烁（0-25%:1次, 25-50%:2次, 50-75%:3次, 75-100%:4次） |

### 3.7 触控板

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 全域触控 | 支持 iPad 全手势 | P0 |
| 全点位触发 | 所有点位均可轻点&点按触发 | P0 |
| 中间区域按压 | 拖动文件时不会掉落 | P0 |
| 防误触机制 | 按键和触摸板可同时使用 | P0 |
| 按压唤醒 | 支持按压唤醒键盘 | P0 |

### 3.8 恢复出厂设置

| 步骤 | 描述 |
|------|------|
| 触发 | 长按 ESC+= 键 3 秒 |
| 指示 | 指示灯闪烁 3 次 |
| 清除 | 清除蓝牙配对记录 |
| 重置 | 恢复默认背光（白光 2 档） |
| 重连 | 重新连接 iPad，背光白光亮起 |

## 7. Agent 评审意见汇总

### 7.1 架构/编码视角

| 维度 | 结论 |
|------|------|
| 技术可行性 | **高** - 现有架构覆盖 90%+ 核心功能 |
