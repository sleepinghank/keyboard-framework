## 7. 原理图关键电路说明

### 7.1 电源管理电路

```
VBUS (USB 5V) → F1 (保险丝) → U3 (过压保护) → U7 (3.3V LDO) → 3V3 (主控供电)
                                       ↓
                                   U4 (充电IC) → BAT (锂电池)
```

### 7.2 开关机电路

```
关机状态:
  - PB19_POWER_EN = 低电平
  - 主控断电

开机流程:
  1. 按下电源键 → POWER_KEY = 低电平
  2. 主控上电启动
  3. PA2 检测到低电平
  4. PB19 输出高电平保持供电
  5. 释放电源键后继续运行

关机流程:
  1. 长按电源键 3 秒
  2. PB19 输出低电平
  3. 主控断电
```

### 7.3 背光驱动电路

```
3V3_LED (电源)
    │
    ├── R (PB0_PWM_R) → 红色 LED 串 → GND
    ├── G (PA12_PWM_G) → 绿色 LED 串 → GND
    └── B (PA13_PWM_B) → 蓝色 LED 串 → GND

背光使能: PB22_BOOT_LED_EN → 控制 3V3_LED LDO (U11)
```
