## 7. 原理图关键电路说明

### 7.1 电源管理电路

```
VHV (USB 5V) → F1 (保险丝) → F2 → U7 (升压/充电控制)
                                    ↓
                              U6 (TP4056 充电IC) → V_BAT (锂电池)
                                    ↓
                              U3 (5.0V LDO) → 5.0V
                                    ↓
                              U4 (3.3V_LED LDO) → 3V3_LED
```

### 7.2 开关机电路

```
关机状态:
  - PB19_POWER_EN = 低电平
  - 主控断电

开机流程:
  1. 按下电源键 → PA2 检测到触发
  2. 主控上电启动
  3. PB19 输出高电平保持供电
  4. 释放电源键后继续运行

关机流程:
  1. 长按电源键 3 秒或合盖
  2. PB19 输出低电平
  3. 主控断电
```

### 7.3 背光驱动电路

```
3V3_LED (电源)
    │
    ├── R (PB0_PWM_R) → 红色 LED 串 → GND
    ├── G (PA12_PWM_G) → 绿色 LED 串 → GND
    └── B (PA13_PWM_B) → 蓝色 LED 串 → GND

背光使能: U4 (3.3V_LED LDO) 由 Q4/Q5 控制
```

### 7.4 充电电路

```
USB VBUS → U1 (EMI滤波) → U5 (USB-C) → U7 (充电控制) → U6 (TP4056) → V_BAT

充电指示:
  - 充电中: LED3 (红) 亮 (TP4056_CHRG = 低)
  - 充满: LED4 (绿) 亮 (TP4056_STDBY = 低)
```
