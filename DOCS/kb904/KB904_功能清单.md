## 5. 蓝牙模块

### 5.1 蓝牙基础参数

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 蓝牙版本 | 蓝牙5.3及以上 | P0 | 硬件 |
| 配对名称 | "inateck NeoMagic Pro" | P0 | 固件 |
| 有效距离 | 10米范围内可实现有效操作 | P0 | 硬件 |
| 通道数量 | 单通道，仅支持连接1台设备 | P0 | 固件 |

### 5.2 蓝牙配对功能

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 首次配对 | 首次开机或恢复出厂后自动进入配对模式 | P0 | 固件 |
| 手动配对 | 长按Fn+Del 3s启动配对模式 | P0 | 固件 |
| 配对超时 | 广播60秒，超时进入休眠 | P0 | 固件 |
| 配对完成时间 | 设备端点击配对后3秒内完成连接 | P0 | 固件 |
| 配对唤醒 | 任意按键唤醒后再次进入广播模式 | P1 | 固件 |

### 5.3 蓝牙回连/轮询功能

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 开机自动回连 | 开机后直接回连上次连接的设备（无需按键） | P0 | 固件 |
| 唤醒自动回连 | 休眠唤醒后自动回连上次连接的设备 | P0 | 固件 |
| 回连完成时间 | 3秒内完成回连 | P0 | 固件 |
| 回连模式时长 | 回连60秒，超时进入休眠 | P0 | 固件 |

### 5.4 蓝牙连接状态处理

| 触发场景 | 键盘行为 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 键盘关闭电源 | 断连，开机后自动回连上次设备 | P0 | 固件 |
| 键盘进入休眠 | 断连，唤醒后自动回连上次设备 | P0 | 固件 |
| 设备关机/关闭蓝牙/超出距离 | 进入回连轮询机制 | P0 | 固件 |
| 设备删除配对记录（键盘开机） | 进入回连轮询机制 | P0 | 固件 |
| 设备删除配对记录（键盘关机） | 再次开机后进入回连模式 | P0 | 固件 |
| 长按配对键后 | 广播60秒，不自动回连，可连接已配对/未配对设备 | P0 | 固件 |
| 配对模式中关机重启 | 回连上一个设备 | P1 | 固件 |
| 配对中取消配对 | 重新进入配对状态 | P1 | 固件 |
| 回连中取消配对 | 重新进入回连机制 | P1 | 固件 |

### 5.5 蓝牙休眠与功耗

| 功能名称 | 简要描述 | 优先级 | 涉及模块 |
|----------|----------|--------|----------|
| 深度休眠进入 | 10分钟无操作进入深度休眠 | P0 | 固件 |
| 休眠唤醒响应 | 休眠状态下触摸板轻触需立即响应 | P0 | 固件 |

## 12. 模块依赖关系

```
硬件层
  ├── Pixart Par2860QN_QFN48 (蓝牙主控)
  ├── Pixart PCT1336QN (触控主控)
  ├── 霍尔开关 (开关机)
  ├── RGB LED (背光)
  ├── 白/蓝/红/绿 LED (指示灯)
  └── 500mAh 锂电池 + 保护电路
        └── 固件层
              ├── 矩阵扫描 ← 按键输入
              ├── 触控板驱动 ← Pixart PCT1336QN
              ├── 蓝牙协议栈 ← Pixart Par2860QN
              ├── 背光驱动 ← RGB LED (PWM)
              ├── 电源管理 ← 电池监测/休眠
              └── OTA 升级 ← 蓝牙通道
                    └── APP层 (inateck origin)
                          ├── 固件升级
                          ├── 自定义键配置
                          ├── ZWSN手势配置
                          ├── 背光时间调整
                          └── 休眠时间调整
```
